# Built-in Tasks


## init

`$ manuel init`

Creates a new `manuelfile` in the current directory

Example:
```bash
$ cd ~/code/my_awesome_project
$ manuel init
>> Creating new manuelfile in /home/username/code/my_awesome_project
```


## list

`$ manuel list`

List all available tasks

Example:
```bash
$ manuel list
build
init
list
manuel_watch
run_tests
start_server
```
